<prism-js>

A Polymer element for syntax highlighting with Prism.js

Usage

Specify language & include code inline

<prism-js language="css"> p { color: blue; } </prism-js>

Include line-numbers

<prism-js language="javascript" linenumbers="true">

if (true) { console.log('foo'); }

Auto-escape HTML for you

<prism-js language="markup" escape="true">

Some paragraph text

Attributes without value

<prism-js language="markup">

<p data-foo></p> <p data-foo ></p>

Name attribute pair

<prism-js language="markup">

<p class=prism></p>

Namespaces

<prism-js language="markup">

<html:p foo:bar="baz" foo:weee></html:p>

Tag that spans multiple lines

<prism-js language="markup">

<p >hello! </p>

Embedded JS and CSS

<prism-js language="markup">

<!DOCTYPE html> <html lang="en"> <head> <meta charset="utf-8" /> <title>I can haz embedded CSS and JS</title> <style> @media print { p { color: red !important; } } </style> </head> <body> <h1>I can haz embedded CSS and JS</h1> <script> if (true) { console.log('foo'); } </script> </body> </html>

LESS variables

<prism-js language="css">

@main-color: red; .foo { background: @main-color; }

Nested strings

<prism-js language="javascript">

var foo = "foo", bar = "He "said" 'hi'!"

NameDescriptionEffect of setting
nameThe value of the <element>'s name attribute.
versionCurrent version of the element.If set, a version number is displayed.
githubGithub project page URL.If set, displays an icon with a link to the repo.
homeHomepage URL.If set, displays an icon with a link to the URL.

This element supports the following markup options. Tags marked as an "x" means you can use any element, but some elements are suggested over others.

Note: You can indicate an element is required by annotating it with a required attribute.

<summary required>
A brief description of what the custom element does.
<pre class="example">
One or more examples of basic usages of the element. This section is hidden if not provided.
<x class="attributes">
Listing of attributes your element accepts. Suggested elements: <table>, <dl>, <li>. This section is hidden if not provided.
<x class="markup">
Section describing <wc-documentation>'s supported child elements (Light DOM that distributes across insertion points). This section is hidden if not provided.
<x class="styling">
Section describing the styling options (custom pseudo elements, CSS Variables) for the element. This section is hidden if not provided.
<x class="api">
Section describing the element's JS API. Children can be marked with the public or private attribute to show their visibility. This section is hidden if not provided.
<x class="events">
Section describing the element's event model. This section is hidden if not provided.
